# Valora

Valora is a memory-powered startup investment intelligence platform built using Hindsight and Groq.

## Features

* Investor Intelligence Analysis
* Startup Pitch Evaluation
* Investor Matchmaking
* Similar Startup Discovery
* Persistent Memory with Hindsight
* Long-Term Learning from Investor Outcomes

## Tech Stack

* Python
* Streamlit
* Hindsight Memory
* Groq LLM

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Project Vision

Valora helps founders learn from historical investor decisions, identify funding risks, discover suitable investors, and improve fundraising outcomes through memory-powered intelligence.
